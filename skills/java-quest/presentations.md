## ステータス画面

「2. ステータスを見る」が選ばれた場合。進捗JSONの値＋ローカル `sessions.log` の集計を組み合わせる。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  {名前}  Lv.{stats.level} {stats.title}
  EXP: {stats.exp} / {stats.exp_to_next}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  攻略済みダンジョン: {数} / {全体数}
  習得スキル: {スキル一覧}
  AI補助: {OFF（自己申告）/ ON（自己申告）/ 未確認}
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  【学習時間】
  今日:     {H時間M分}
  今週:     {H時間M分}（{セッション数}回）
  累計:     {H時間M分}

  よく訪れたダンジョン（累計）:
    1. C1-01 プログラムの構造   45分
    2. C1-02 標準出力           30分
    3. C1-03 変数と型（数値）   15分
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- AI補助の表記ルールは「おかえり挨拶」と同じ（`settings.ai_assist_disabled` を参照）
- 学習時間の算出ロジックは `session-tracking.md` を参照

## 職業画面

「3. 職業を確認する」が選ばれた場合。

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  【基本職】
  剣士      ★マスター済み
  武闘家    ●●●●○○ (4/6)

  【上級職】
  狂戦士  （前提: 剣士+武闘家）
    → 未解放
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

- マスター済み → `★マスター済み`
- 進行中 → `●` × 取得済みスキル数 ＋ `○` × 残り、括弧内に `(取得済み/必要数)`
- 上級職は前提の基本職名を `（前提: A+B）` 形式で明記し、解放状態を `→ 未解放` / `→ マスター済み` などで示す

## レベルアップ演出

EXP加算後にレベルアップ条件を満たした場合:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  レベルが上がりました！

  Lv.{旧} → Lv.{新}
  新たな称号: 【{称号}】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

直後にリーナが祝福のセリフを述べる（`dialogues.md` の「レベルアップ」参照）。

レベル・称号更新も進捗JSON の `stats.level` / `stats.title` / `stats.exp_to_next` に反映し、**即時 PUT する**。
