## 2. レベルテーブル設計方針

### 2.1 最大レベルの決定

- 全ダンジョンクリア時の平均EXP = 約6,900
- 最大レベルの累計EXPは平均プレイで到達できる水準に設定する
- 最終レベルはS評価を多く取った優秀なプレイヤーが到達する「ご褒美レベル」とする
- **最大レベル: Lv25**（累計EXP 6,750 — 平均プレイで到達可能）

### 2.2 EXPカーブの設計

必要EXPの増加カーブは以下の方針で設計する:

- **Lv1〜5**: 序盤は緩やかに。100 → 150 → 200 → 250 → 300（既存テーブル）
- **Lv6〜10**: 中盤前半。+50ずつ増加を維持: 350 → 400 → 500 → 600（既存）
- **Lv11〜15**: 中盤後半。増加幅を緩やかに拡大
- **Lv16〜20**: 終盤前半。安定した増加
- **Lv21〜25**: 終盤。最終レベルに向けて収束

増加式（目安）:
```
Lv1-5:   base=100, 差分+50/Lv
Lv6-10:  差分+50〜100/Lv（既存テーブル通り）
Lv11-15: 差分+25〜50/Lv（増加が緩やかに）
Lv16-20: 差分+25/Lv（安定）
Lv21-25: 差分+0〜25/Lv（収束）
```

### 2.3 検証

- Lv25到達（累計6,750）: 平均プレイ（6,910）でギリギリ到達 → 適切
- Lv20到達（累計4,250）: 最小プレイ（5,555）でも余裕 → 適切
- Lv10到達（累計2,850）: カテゴリ1完了時の平均EXP（2,575）でほぼ到達 → 適切
- 最大EXP(8,265)でもLv25で打ち止め → 余剰EXPは1,500程度。将来のLv拡張余地あり

## 6. 今後の課題

- カテゴリ2〜6のEXP設定値は推定。正式な設計書更新時に確定する
- 復習モードやサブクエストによるEXP追加を導入した場合、Lv上限の再検討が必要
- 将来の拡張ダンジョン追加時には、Lv26以降のテーブル拡張を行う
