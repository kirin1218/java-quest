# プライバシーポリシー

最終更新日: 2026-04-15

本ポリシーは、Claude Code スキル **java-quest**（以下「本スキル」）の利用にあたって取得・保存する情報の取り扱いを定めるものです。本スキルを利用された時点で、本ポリシーに同意いただいたものとみなします。

---

## 1. 取得する情報

本スキルは、学習進捗管理を目的として以下の情報を取得・保存します。

| 項目 | 取得経路 | 保存先 |
|------|---------|-------|
| 冒険者名（ニックネーム） | 初回起動時に冒険者自身が入力 | `~/.config/java-quest/config.yaml`（ローカル）／配信 API（サーバー） |
| 冒険者番号（UUID v4） | 初回起動時に自動発行 | ローカル＋サーバー |
| 学習進捗（攻略済みダンジョン・EXP・レベル・職業マスター状況等） | スキル利用に伴い自動更新 | ローカル＋サーバー |
| 開発ソースのルートフォルダパス | 初回起動時に冒険者自身が入力 | ローカルのみ |
| API アクセスログ（IP アドレス・タイムスタンプ・リクエスト内容） | サーバー側で自動記録 | サーバーのみ |

**取得しない情報**: 本名、メールアドレス、電話番号、住所、決済情報、Google/GitHub 等の外部アカウント情報は一切取得しません。

## 2. 利用目的

取得した情報は、以下の目的にのみ利用します。

- 冒険者個人の学習進捗の保存・復元
- 障害対応・不正利用調査（API アクセスログ）

上記以外の目的には利用しません。

## 3. 第三者提供

取得した情報を第三者に提供することはありません。法令に基づく開示請求があった場合を除きます。

## 4. 保存期間

- 継続利用中は保管します。
- 削除請求を受領した場合、合理的な期間内（原則 30 日以内）に該当データを消去します。

## 5. 削除請求・問い合わせ窓口

削除請求およびプライバシーに関するお問い合わせは、以下の GitHub Issue にて受け付けます。

- https://github.com/kirin1218/java-quest/issues

削除請求の際は、対象の冒険者番号（UUID）を本文に記載してください（冒険者名は記載不要です）。

## 6. EU 居住者（GDPR）への対応

EU 一般データ保護規則（GDPR）第 13 条・第 17 条等に基づく以下の権利行使を受け付けます。

- アクセス権・訂正権・削除権（忘れられる権利）・処理制限権・データポータビリティ権・異議申立権

行使方法は第 5 項と同じく GitHub Issue にてご連絡ください。

なお、本スキルの管理者は EU 域内に拠点を持たず、EU 居住者を主たる対象としていません。GDPR の適用がある場合に限り、上記の権利行使に応じます。

## 7. データの管理体制

- サーバーは AWS（東京リージョン）にて運用しています。
- 通信は HTTPS で暗号化されます。
- 管理者: Tsuyoshi Hemmi（個人）

## 8. ポリシーの変更

本ポリシーは、法令の変更や本スキルの仕様変更に伴い、予告なく改定する場合があります。改定後のポリシーは本ファイルの「最終更新日」をもって発効するものとします。
